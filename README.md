# amazon-in-quicken
A vibe-coded skill that automates adding product descriptions to Quicken for anything purchasedon Amazon.  
Created completely in the ChatGPT Mac app.  I didn't write a single line of code.  
The skill isn't easily run by anyone else, since it requires a Chase Visa account and a Quicken Classic Premier subscription.  
But the whole point of this skill is to show that people can make their own, idiosyncratic automations that no professional programmer would ever create as one person's custom app.
